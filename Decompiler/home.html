Decompiler Tool

When using LLM's, it is often difficult and time consuming to verify what the code does, find bugs, and generally comprehend the structure of the code. The decompiler tool is useful to get a quick mental picture of code generated by an LLM, or any code that a developer is unfamiliar with. It accomplishes this by taking code as an input, and generating pseudocode or block diagrams as the output. I'm referring to this as a decompiler, because it works in the opposite direction of a compiler, generating something more readable by a human from machine code that takes time and effort to comprehend. I used Mermaid to build this.
